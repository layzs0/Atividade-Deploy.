# ⚽Atividade Deploy - Cadastro de Jogadores 

Este proketo consiste em um sistema de cadastro de jogadores de futebol, desenvolvido como parte de uma atividade de deploy no SENAI - Serviço Nacional de Aprendizagem Industrial. O repositório está organizado em branches para separar as responsabilidades do sistema.

📂Estrutura do Repositório
O projeto utiliza branches específicas para cada parte da aplicação:

- main: Contém a documentação principal.

- Tcc-Front: Código fonte da interface (React/Node).

- Tcc-Back: Código fonte da API e banco de dados.

🚀 Como Executar o Projeto
Para rodar o projeto localmente, você precisará baixar os arquivos de cada branch ou clonar o repositório.

Configurando o Front-end
Mude para a branch do front: git checkout Tcc-Front

1.Instale as dependências:

'''Bash
npm install'''
Inicie a aplicação:

'''Bash
npm start'''
2. Configurando o Back-end
Mude para a branch do back: git checkout Tcc-Back

Instale as dependências:

'''Bash
npm install'''
Inicie o servidor:

'''Bash
npm run dev'''

🛠️ Tecnologias Utilizadas

- Frontend: React.js / CSS
- Backend: Node.js / Express
- Deploy: (Cite aqui onde você fez o deploy, ex: Vercel, Netlify ou Render)
