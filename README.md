# ⚽Atividade Deploy - Cadastro de Jogadores 

Este projeto consiste em um sistema de cadastro de jogadores de futebol, desenvolvido como parte de uma atividade de deploy no SENAI - Serviço Nacional de Aprendizagem Industrial. O repositório está organizado em branches para separar as responsabilidades do sistema.

📂Estrutura do Repositório
O projeto utiliza branches específicas para cada parte da aplicação:

- main: Contém a documentação principal.

- Tcc-Front: Código fonte da interface (React/Node).

- Tcc-Back: Código fonte da API e banco de dados.

🚀 Como Executar o Projeto

Para rodar o projeto localmente, você precisará baixar os arquivos de cada branch ou clonar o repositório.

🎨Configurando o Front-end:

1.Mude para a branch do front: git checkout Tcc-Front

2.Instale as dependências:

___
Bash
npm install
___

3.Inicie a aplicação:

___
Bash 
npm start
___

🗄️Configurando o Back-end

1.Mude para a branch do back: git checkout Tcc-Back

2.Instale as dependências:

___
bash 
npm install
___

3.Inicie o servidor:

___
bash
npm run dev
___

🛠️ Tecnologias Utilizadas

- Frontend: React.js/ Css / javaScript / Html
- Backend: Node.js
